# P19 - Learning Implicit User Profiles for Personalized Dialog 论文笔记

**标题**: Learning Implicit User Profiles for Personalized Retrieval-Based Dialog Systems  
**发表**: 2021  

---

## 1. 核心贡献

- 从用户对话历史中**隐式学习**用户偏好
- 无需显式标注，通过对话行为推断用户画像
- 提升对话系统的个性化回复选择

---

## 2. 方法要点

| 组件 | 说明 |
|------|------|
| 隐式画像构建 | 从对话中的选择行为提取偏好信号 |
| 偏好建模 | 用户对特定类型回复的选择概率 |
| 个性化排序 | 基于画像对候选回复重新排序 |

---

## 3. 与我们研究的关联

| 维度 | 启示 |
|------|------|
| **隐式偏好学习** | 与 Odin 的 Personalizer 互补：Odin 需要显式反馈，此方法支持隐式学习 |
| **用户画像初始化** | 冷启动时可先从对话行为推断初始偏好 |
| **与 P07 结合** | Odin 的显式反馈 + 隐式学习 = 更完善的用户层消歧 |
