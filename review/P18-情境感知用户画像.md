# P18 - Context-Aware User Profiling 论文笔记

**主题**: Context-Aware User Profiling  
**来源**: 综合性研究主题（非单篇论文）

---

## 1. 核心概念

- **动态用户画像**: 根据当前情境动态调整用户模型
- **情境感知**: 考虑时间、地点、设备、历史行为等情境因素
- **个性化服务**: 基于画像提供差异化服务

---

## 2. 与我们研究的关联

| 维度 | 启示 |
|------|------|
| **用户层消歧** | 用户画像可用于预测用户在歧义中的偏好 |
| **情境因素** | 同一用户在不同场景下可能有不同歧义解释偏好 |
| **动态适配** | 用户偏好不是静态的，需要持续更新 |
