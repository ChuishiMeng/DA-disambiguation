# P11 - Disambiguation in Conversational QA in the Era of LLMs 论文笔记

**标题**: Disambiguation in Conversational Question Answering in the Era of LLMs and Agents: A Survey  
**发表**: EMNLP 2025  
**作者**: Tanjim 等  

---

## 1. 核心贡献

- **综述论文**：系统梳理 LLM 时代对话式问答中的消歧方法
- 将现有消歧方法分为三大类：
  1. **Query Rewriting**（查询改写）：改写歧义查询为无歧义查询
  2. **Long-Form Answer Generation**（长答案生成）：直接生成覆盖多种解释的答案
  3. **Asking Clarifying Questions**（澄清问题）：与用户交互消歧

---

## 2. 关键发现

| 发现 | 说明 |
|------|------|
| LLM 消歧不完美 | 即使强大 LLM 也难以检测细粒度歧义 |
| 三类方法互补 | Query Rewriting 适合简单歧义，Clarifying Questions 适合复杂歧义 |
| Agent 新范式 | 多 Agent 协作正在成为新的消歧范式 |
| 缺乏统一框架 | 各方法独立工作，缺乏统一消歧框架 |

---

## 3. 与我们研究的关联

| 维度 | 启示 |
|------|------|
| **方法分类** | 三类消歧方法的分类可作为我们方法选择的理论基础 |
| **统一框架缺失** | 直接验证了我们的研究方向：构建统一消歧框架 |
| **Agent 范式** | 多 Agent 消歧是前沿趋势，可在我们的框架中引入 |
| **Text-to-SQL 缺失** | 该综述未专门覆盖 Text-to-SQL 场景，说明 Text-to-SQL 消歧是未被充分探索的方向 |
