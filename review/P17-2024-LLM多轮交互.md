# P17 - Multi-Turn Interactions for Text-to-SQL with LLMs 论文笔记

**标题**: Multi-Turn Interactions for Text-to-SQL with Large Language Models  
**发表**: arXiv 2408.11062 (2024)  
**来源**: Zuoyebang Education Technology  

---

## 1. 核心贡献

- 研究 LLM 在多轮 Text-to-SQL 交互中的能力
- 提出动态基准测试评估多轮交互质量
- 分析 LLM 在多轮场景下的退化模式

---

## 2. 方法要点

| 组件 | 说明 |
|------|------|
| 多轮交互设计 | LLM 主动追问澄清 + 用户回复 + SQL 更新 |
| 动态评估 | 不仅评估最终 SQL，还评估交互过程质量 |
| 退化分析 | LLM 在长对话中容易丢失早期上下文 |

---

## 3. 与我们研究的关联

| 维度 | 启示 |
|------|------|
| **LLM 多轮退化** | 多轮消歧时需防止上下文丢失，可能需要状态压缩 |
| **交互质量评估** | 不只评估 SQL 准确率，也评估交互效率（轮次、用户满意度） |
| **澄清策略** | LLM 主动追问澄清的时机和方式需要优化 |
