# P12 - Dynamic Knowledge Adaptation for Domain-Specific Text-to-SQL 论文笔记

**标题**: Dynamic Knowledge Adaptation for Domain-Specific Text-to-SQL  
**发表**: Applied Sciences (MDPI) 2025  

---

## 1. 核心贡献

- 领域知识的动态适配方法
- 将业务规则与 SQL 逻辑对齐
- 通过知识注入提升垂直领域 Text-to-SQL 准确率

---

## 2. 方法要点

| 组件 | 说明 |
|------|------|
| 知识提取 | 从业务文档/规则中提取领域知识 |
| 动态适配 | 根据查询内容动态选择相关领域知识 |
| SQL 对齐 | 确保生成的 SQL 符合业务语义 |

---

## 3. 与我们研究的关联

| 维度 | 启示 |
|------|------|
| **业务层歧义** | 直接对应我们的"业务层"消歧：业务概念的定义模糊 |
| **知识动态选择** | 不同查询需要不同业务知识，需要动态适配机制 |
| **知识-SQL 对齐** | 确保业务知识正确映射到 SQL 逻辑 |
| **与 P13/P14 互补** | 三篇共同构成领域知识注入的技术栈 |
