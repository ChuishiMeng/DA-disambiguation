## 一、核心文献（4篇）

### P01 - Disambiguate First, Parse Later (2025)

| 项目 | 内容 |
|------|------|
| **标题** | Disambiguate First, Parse Later: Generating Interpretations for Ambiguity Resolution in Semantic Parsing |
| **作者** | Irina Saparina, Mirella Lapata |
| **发表** | ACL 2025 Findings |
| **链接** | https://arxiv.org/abs/2502.18448 |
| **PDF** | https://aclanthology.org/2025.findings-acl.863.pdf |

**核心贡献**：
- 提出两阶段方法：先生成自然语言解释，再解析为 SQL
- 解决 Text-to-SQL 中的歧义问题
- 模块化设计，解耦歧义识别和 SQL 生成

### P02 - AMBROSIA (2024)

| 项目 | 内容 |
|------|------|
| **标题** | AMBROSIA: A Benchmark for Parsing Ambiguous Questions into Database Queries |
| **作者** | Irina Saparina, Mirella Lapata 等 |
| **发表** | NeurIPS 2024 |
| **链接** | https://arxiv.org/abs/2406.19073 |
| **官网** | https://ambrosia-benchmark.github.io/ |

**核心贡献**：
- 歧义类型分类：**scope ambiguity（范围歧义）**、**attachment ambiguity（依附歧义）**、**vagueness（模糊性）**
- 构建了歧义问题到 SQL 的基准数据集

| **作者** | Ding, Lin 等 |
| **发表** | arXiv 2508.15276 |
| **链接** | https://arxiv.org/abs/2508.15276 |
