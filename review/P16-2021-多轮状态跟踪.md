# P16 - Tracking Interaction States for Multi-Turn Text-to-SQL 论文笔记

**标题**: Tracking Interaction States for Multi-Turn Text-to-SQL Semantic Parsing  
**发表**: AAAI 2021  
**来源**: 这是 CoSQL 数据集上的基础工作

---

## 1. 核心贡献

- 提出 **交互状态跟踪** 机制处理多轮 Text-to-SQL
- 维护对话状态（已选表/列、过滤条件、历史 SQL 片段）
- 在 CoSQL 数据集上验证有效性

---

## 2. 方法要点

| 组件 | 说明 |
|------|------|
| 状态编码 | 将历史对话和中间 SQL 编码为状态向量 |
| 状态更新 | 每轮对话后更新状态 |
| 上下文继承 | 后续轮次继承前轮的 Schema 选择和约束 |

---

## 3. 与我们研究的关联

| 维度 | 启示 |
|------|------|
| **状态跟踪** | 多轮消歧需要维护消歧状态（已消歧项、待消歧项） |
| **上下文继承** | 已消歧的项应在后续对话中保持 |
| **与 P17 互补** | P16 是传统方法，P17 是 LLM 时代的多轮方法 |
